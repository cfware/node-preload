Request that Node.js child processes preload modules

### Install node-preload

This module requires node.js 8 or above.

```sh
npm i node-preload
```

## Usage

```js
'use strict';

require('node-preload');
```

## Running tests

Tests are provided by xo and ava.

```sh
npm install
npm test
```
